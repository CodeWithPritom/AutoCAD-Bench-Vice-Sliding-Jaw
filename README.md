# Bench Vice Sliding Jaw - 3D Modeling & Drafting

## 📌 Project Overview
This repository contains the design, 3D modeling, and technical drafting of a **Bench Vice Sliding Jaw**. The project demonstrates technical proficiency in AutoCAD, focusing on geometric complexity, mechanical accuracy, and standardized engineering documentation.

## 🛠️ Skills & Tools Demonstrated
- **Software:** AutoCAD 2024/2025
- **3D Modeling:** Solid primitives, Extrusion, and Union operations.
- **Boolean Logic:** Complex subtraction for mechanical slots and holes.
- **Drafting:** 3rd Angle Orthographic Projections & Isometric Views.
- **Technical Documentation:** Annotation, Layer Management, and Dimension Styles.

## 📂 Project Structure
- **/DWG_Files:** Contains the source files for 3D Model, Isometric View, and Projections.
- **/PDF_Report:** A comprehensive document featuring all views and descriptions.
- **/Screenshots:** High-resolution renders and technical drafts.

## 📐 Technical Highlights
### 3D Modeling
The model was built using a bottom-up approach, ensuring structural integrity. Key features include:
- Realistic screw housing holes.
- Precision-engineered sliding grooves.
- Filleted edges for a professional finish.

### Drafting & Projections
To meet industrial standards, the following views were generated:
1. **Front View:** Showcasing the primary mechanical profile.
2. **Top View:** Highlighting the sliding interface.
3. **Side View:** Detailing the thickness and depth.
4. **Isometric View:** Providing a 3D perspective in a 2D plane.

## 📝 Description
The sliding jaw is a critical component of a bench vice, responsible for gripping workpieces. This model simulates the actual mechanical part, incorporating all necessary slots and holes required for its functionality.
